# Tea and Coffee Vending Machine System

A simple frontend-based **Tea and Coffee Vending Machine System** built using **HTML, CSS, and JavaScript**.  
This project simulates a vending machine where users can select tea or coffee with or without sugar, and the system manages stock and calculates sales and profit.

---

# Features

- Select drinks:
  - Tea with Sugar
  - Tea without Sugar
  - Coffee with Sugar
  - Coffee without Sugar
- Automatic stock management
- Stock reduces after each sale
- Displays:
  - Total sales
  - Total cost

- Simple and user-friendly UI
